一、檔案/資料夾及其性質：
1.Game.java:主程式檔，創建遊戲視窗，以及更新圖鑑的method。
2.Label.java:從不同的資料夾中抓取圖檔的method、抓取使用者圖鑑中圖片的method，以及將已抽到的圖卡存進圖鑑的method。
3.Majang.java:設定麻將牌的類別。
4.Shuffle.java:洗牌，以及創建出所有麻將牌。
5.Player.java:包含玩家的手牌，以及抽牌、出牌的method，和判斷吃、碰的method。
6.PlayerPanel.java:放置玩家手牌和牌池中麻將牌的panel，以及更新panel的method。
7.Work.java:執行3位電腦玩家的摸牌以及出牌，並且判斷使用者能否吃、碰，或是胡牌、流局。
8.PumpCard.java:抽取SDGs圖卡的panel。
9.Result.java:胡牌或流局結束畫面的panel。
10.card.txt:存取玩家所抽取到的所有圖卡。
11.card(卡牌圖檔資料夾):SDGs圖卡和台灣小吃圖卡的圖片。
12.picture(麻將牌圖檔資料夾):麻將牌、背景和按鈕的所有圖片。

二、開發軟體工具安裝之順序步驟與設定:
1.安裝JDK
2.設定系統環境變數
3.安裝Visual Studio Code

三、自行開發程式碼之安裝設定：
1.安裝9個java檔、1個txt檔，以及2個資料夾

四、測試安裝成功之步驟：
1.編譯程式碼javac Game.java
2.如系統沒報錯，即安裝成功。
3.執行檔案java Game
